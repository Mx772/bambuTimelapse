Settings

Configure printer, camera, and timelapse options

Found on the printer label or in the Bambu app.

Settings → Network → LAN Only Mode on your printer.

Connection

Saves settings and connects to the printer's MQTT broker on port 8883 over TLS. Enable Developer Mode on the printer if connection fails.

Credentials in the URL are supported. Use TCP transport for reliability.

frames per second

24 fps is standard. Lower = slower/longer video, higher = faster.

every N layers

Set to 1 to capture every layer, 2 for every other layer, etc.

IANA timezone ID used for all date/time display and the ETA in the browser tab. Examples: America/Chicago, Europe/London, Asia/Tokyo. Match this to the TZ environment variable in your Docker config.
